# Student Management System (Java)

## Description
This is a Java-based application to manage student records using a simple GUI.  
It allows users to perform basic operations like adding, viewing, searching, updating, and deleting student details.

##  Features
- Add Student
- View All Students
- Search Student by ID
- Update Student Details
- Delete Student
- Data stored using file handling (students.txt)

## Technologies Used
- Java
- Swing (GUI)
- File Handling

## How to Run
1. Compile the files:
   javac *.java

2. Run the application:
   java StudentGUI

##  Project Structure
- Student.java
- StudentManager.java
- StudentGUI.java
- students.txt
